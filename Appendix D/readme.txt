This directory contains the itcsim library, as well as several scripts used to compare different models of cooperativity or examine their ability to fit available experimental data.

./comparison: Python scripts to compare the partition functions of the Saroff-Kiefer and Kleckner models, as well as validate models implemented in itcsim

./data: Experimental data, obtained from Ian Kleckner, used to test models

./examples: Example ITC fitting scripts that use the itcsim library

./itcsimlib: The hybrid C and Python itcsim library
