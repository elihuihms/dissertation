This directory contains experimental files and scripts relating for processing and analysis of NMR, SAXS, and sedimentation data.

./nmr: Contains XPlor-NIH and experimental data for restraining/refining the AT trimer structure.

./saxs: Contains experimental SAXS profiles, formfactor file, and OLIGOMER input/execution script.

./sedimentation: Contains a list of calculated experimental concentrations, and scripts for processing sedimentation residuals from SEDPHAT output.
